# Dashboard Blog

## O que é
Dashboard simples em HTML/CSS/JS puro para visualizar métricas de artigos do WordPress.

## Regras
- HTML, CSS e JavaScript puro (sem frameworks, sem instalações)
- Consumir dados via WordPress REST API (wp-json/wp/v2/posts)
- Comentários em português
- Layout simples e limpo, funciona no navegador direto (sem servidor)
- Não usar bibliotecas externas além de Chart.js para gráficos
